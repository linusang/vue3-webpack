# A starter kit for Vue 3 + Webpack 5

## Features

- TypeScript
- sass
- source maps

## Todo

- unit test
- tailwindcss

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Lint using ESLint

```
npm run lint
```

### Fix Lint using ESLint

```
npm run lint:fix
```

### Build for production (dist folder)

```
npm run build
```

### Preview production locally

```
npm run preview
```
